# To-Do List Assignment (Browser Storage)

## Project Description
This project is a simple To-Do List web application built with HTML, CSS, and JavaScript.  
It demonstrates the use of **Browser Storage (Local Storage)** to store tasks so that they remain available even after refreshing the page.

## Features
- Add a new task
- Mark a task as completed
- Delete a task
- Tasks persist using Local Storage
- Clean and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Browser Local Storage

## How It Works
Tasks are saved in the browser using `localStorage`.  
When the page loads, saved tasks are retrieved and displayed automatically.
